Documentation for OpenWARP was developed by TopCoder during the OpenWARP project. The following documents are available:

  • OpenWARP Deployment Guide: Description of how to install and run OpenWARP.
  • OpenWarp ECN Documentation and Theory: Updates to the OpenWARP theory documentation so that it includes feature additions performed as part of the OpenWARP project.
  • OpenWARP Installer Build Guide: Description of how to build and deploy the binary installers for Mac, Linux, and Windows.
